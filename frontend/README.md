# Frontend

## Week 1 Scope
Scaffolding, folder conventions, and mock-data screen spikes.
No API integration yet.

## Run locally
1. cd frontend
2. npm install
3. npm run dev

## Where to work
- Pages: src/pages/
- Components: src/components/
- Mock data: src/mock/
- Styles: src/index.css (global for Week 1)
